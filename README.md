## English

Ah, you've stumbled upon my Album Cover Gallery. Yes, there are a lot of album covers. No, I don't have a problem.

### What is this?

A collection of AI-generated album covers in various styles:
Lofi, Minimal, Collage, Psychedelic, Sketch, Gothic, Nature, Glitch, Ugly Design, and Zen.

Because apparently, I had too much free time.

### The Styles

| Style | Vibe | When to Use |
|-------|------|------------|
| Lofi | Rainy window, coffee, depression | When you're pretending to study |
| Minimal | "I'm an intellectual" | When plainwhitewallpapers isn't enough |
| Collage | Magazine cutouts, chaos | When one image isn't confusing enough |
| Psychedelic | DO NOT USE BEFORE DRUGS* | *Just kidding, please don't do drugs |
| Sketch | Hand-drawn, "I could do that" | When stock photos feel too real |
| Gothic | Dark, mysterious | When you're going through something |
| Nature | Trees, mountains, inner peace | When you need to pretend to be healthy |
| Glitch | Corrupted, broken | When the vinyl got stuck |
| Ugly Design | Bold, offensive | When Helvetica just won't cut it |
| Zen | Empty, nothingness | When you realize life's meaningless |

### Features

- **Many Styles × Many Covers = Endless Possibilities**
- **Lazy Loading** — Because patience is a virtue
- **Lightbox** — Click to see them full-size. You're welcome.
- **Filter** — Find exactly the style of procrastination you need today

### Tech Stack

```
HTML + CSS + JS + Existential Crisis + AI
```

Built with vanilla JS because frameworks are for people who can't handle their own callbacks.
